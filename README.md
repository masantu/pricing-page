# A fun pricing page 🎈

A fun pricing page with Gifs 🎞🖼.

Made with typescript Vue and tailwindcss.


[Live demo](https://distracted-mayer-08c583.netlify.app/)


## Project setup 🚀
```
npm install
```

### Compiles and hot-reloads for development 👩‍💻
```
npm run serve
```

### Compiles and minifies for production 📦
```
npm run build
```
